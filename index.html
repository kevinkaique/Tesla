Tesla MODEL 3

O Model 3 é um automóvel sedã médio-grande elétrico, produzido pela Tesla. Chegou a chamar-se Tesla Model E, porém depois de uma disputa com a Ford pelo nome acabou por se chamar Tesla Model 3. O Model 3 foi revelado no dia 31 de março de 2016, num evento organizado pela Tesla em Hawthorne, Califórnia.
